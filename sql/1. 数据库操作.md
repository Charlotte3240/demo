## 1. 数据库连接

```bash
mysql -u root -p
# 输入密码

# -P 端口 -h ip地址 
mysql -u root -p -P 3306 -h localhost
```

## 2. 创建数据库

```bash
# create database 数据库名称 charset 字符集
create database sellSys charset utf8mb4;
```

## 3. 切换数据库

```bash
# use 数据库名称
use sellSys;
```

## 4. 删除数据库

```bash
# drop database 数据库名称
drop database sellSys;

# 如果数据库不存在可能会报错，优化成如果村子啊
drop database if exists sellSys;
```

## 5. 查看创建数据库的结果

```bash
mysql> show create database sellSys
    -> ;
```
